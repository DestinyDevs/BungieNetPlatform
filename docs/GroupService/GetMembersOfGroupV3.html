GroupService / GetMembersOfGroupV3

Edit Wiki

Info

Returns a list of members for a given group.

  • URI: /Group/{groupId}/MembersV3/
  • Method: GET
  • Accessibility: Public
  • Service: GroupService

Parameters

Path Parameters

Name Description
groupId A valid groupId.

Query String Parameters

Name Description
itemsPerPage Items per page. Default is 10.
currentPage The current page. Starts at 1. Required.
memberType Default is -1. -1=All Members, 0=Member, 1=Admin, 2=Founder
platformType A valid clan membership type. Default is 0. 0=None, 1=Xbox, 2=PSN
sort Default is 0.
nameSearch Not working?

JSON POST Parameters

None

https://www.bungie.net/Platform/Group/177526/MembersV3/?currentPage=1
