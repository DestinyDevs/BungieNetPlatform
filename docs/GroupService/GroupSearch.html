GroupService / GroupSearch

Edit Wiki

This page was generated with Wiki Builder. Do not change the format!

Info

Searches for groups matching input string. Returns matching group information.

  • URI: /Group/Search/
  • Method: POST
  • Accessibility: Public
  • Service: GroupService

Parameters

Path Parameters

None

Query String Parameters

Name Description
populatefriends true or false.

JSON POST Parameters

Name Description
contents.searchValue Group name to search for.
contents.searchType Filter by group type. Default 0.
creationDate  
currentPage The current page to return. Starts at 1.
groupMemberCountFilter Filter groups based on how many members they have.
itemsPerPage Number of results per page. Default is 10.
localeFilter  
membershipType A valid Bungie.net membershipType.
sortBy Default 0.
tagText

POST https://www.bungie.net/Platform/Group/Search/
