GroupService / DisableClanForGroup

Info

Disables clan features for a group that the current user is an admin for.

  • URI: /Group/{groupId}/Clans/Disable/{clanMembershipType}/
  • Method: POST
  • Accessibility: Private
  • Service: GroupService

Parameters

Path Parameters

Name Description
groupId A valid groupId.
clanMembershipType A valid clan membership type. 1=Xbox, 2=PSN

Query String Parameters

None

JSON POST Parameters

None

Example

POST https://www.bungie.net/Platform/Group/177526/Clans/Disable/2/
