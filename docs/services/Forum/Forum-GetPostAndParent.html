Forum.GetPostAndParent

A community run wiki for the Bungie.net Platform APIs.

Source

This page was generated with Wiki Builder. Do not change the format!

Info

Returns the post specified and its immediate parent.

  • URI: /Forum/GetPostAndParent/{childPostId}/
  • Basepath: https://bungie.net/Platform
  • Method: GET
  • Service: Forum
  • Permissions: None
  • Officially Supported: Yes

Parameters

Path Parameters

Name Schema Required Description
childPostId integer:uint64 Yes  

Query String Parameters

Name Schema Required Description
showbanned string No If this value is not null or empty, banned posts are requested to be returned

Example

Request

GET https://bungie.net/Platform/Forum/GetPostAndParent/{childPostId}/
