Forum.GetPoll

A community run wiki for the Bungie.net Platform APIs.

Edit Wiki

Info

Gets the specified forum poll.

  • URI: /Forum/Poll/{topicId}/
  • Basepath: https://bungie.net/Platform
  • Method: GET
  • Service: Forum
  • Permissions: None
  • Officially Supported: Yes

Parameters

Path Parameters

Name Schema Required Description
topicId integer:int64 Yes The post id of the topic that has the poll.

Query String Parameters

None

Example

Request

GET https://bungie.net/Platform/Forum/Poll/{topicId}/
