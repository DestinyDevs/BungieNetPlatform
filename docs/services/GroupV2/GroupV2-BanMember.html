GroupV2.BanMember

A community run wiki for the Bungie.net Platform APIs.

Info

Bans the requested member from the requested group for the specified period of time.

  • URI: /GroupV2/{groupId}/Members/{membershipType}/{membershipId}/Ban/
  • Basepath: https://www.bungie.net/Platform
  • Method: POST
  • Service: GroupV2
  • Permissions: AdminGroups
  • Officially Supported: Yes

Parameters

Path Parameters

Name Schema Required Description
groupId integer:int64 Yes Group ID that has the member to ban.
membershipId integer:int64 Yes Membership ID of the member to ban from the group.
membershipType BungieMembershipType:Enum Yes Membership type of the provided membership ID.

Query String Parameters

None
