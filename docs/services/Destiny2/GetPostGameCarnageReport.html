Destiny2.GetPostGameCarnageReport

A community run wiki for the Bungie.net Platform APIs.

Info

Gets the available post game carnage report for the activity ID.

  • URI: /Destiny2/Stats/PostGameCarnageReport/{activityId}/
  • Basepath: https://bungie.net/Platform
  • Method: GET
  • Service: Destiny2
  • Permissions: None
  • Officially Supported: Yes

Parameters

Path Parameters

Name Schema Required Description
activityId integer:int64 Yes The ID of the activity whose PGCR is requested.

Query String Parameters

None

Request

GET https://bungie.net/Platform/Destiny2/Stats/PostGameCarnageReport/{activityId}/
