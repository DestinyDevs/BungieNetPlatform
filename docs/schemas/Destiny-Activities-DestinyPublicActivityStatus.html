Destiny.Activities.DestinyPublicActivityStatus

A community run wiki for the Bungie.net Platform APIs.

Info

Represents the public-facing status of an activity: any data about what is currently active in the Activity, regardless of an individual character's progress in it.

  • Type: object

Properties

Name Type Description
challengeObjectiveHashes integer:uint32[] Active Challenges for the activity, if any - represented as hashes for DestinyObjectiveDefinitions.
modifierHashes integer:uint32[] The active modifiers on this activity, if any - represented as hashes for DestinyActivityModifierDefinitions.
rewardTooltipItems Destiny.DestinyItemQuantity[] If the activity itself provides any specific "mock" rewards, this will be the items and their quantity.

Why "mock", you ask? Because these are the rewards as they are represented in the tooltip of the Activity.

These are often pointers to fake items that look good in a tooltip, but represent an abstract concept of
what you will get for a reward rather than the specific items you may obtain.

Example

{
    // Type: integer:uint32[]
    "challengeObjectiveHashes": [
       // Type: integer:uint32
        0
    ],
    // Type: integer:uint32[]
    "modifierHashes": [
       // Type: integer:uint32
        0
    ],
    // Type: Destiny.DestinyItemQuantity[]
    "rewardTooltipItems": [
       // Type: Destiny.DestinyItemQuantity
        {
            // Type: integer:uint32
            "itemHash": 0,
            // Type: integer:int64:nullable
            "itemInstanceId": 0,
            // Type: integer:int32
            "quantity": 0
        }
    ]
}
