Destiny.Milestones.DestinyPublicMilestoneQuest

Info

  • Type: object

Properties

Name Type Description
questItemHash integer:uint32 Quests are defined as Items in content. As such, this is the hash identifier
of the DestinyInventoryItemDefinition that represents this quest. It will have pointers to all of the steps
in the quest, and display information for the quest (title, description, icon etc)
Individual steps will be referred to in the Quest item's DestinyInventoryItemDefinition.setData
property, and themselves are Items with their own renderable data.
activity Destiny.Milestones.DestinyPublicMilestoneActivity A milestone need not have an active activity, but if there is one it will be returned here,
along with any variant and additional information.
challenges Destiny.Milestones.DestinyPublicMilestoneChallenge[] For the given quest there could be 0-to-Many challenges: mini quests
that you can perform in the course of doing this quest, that may grant you rewards and benefits.

{
    // Type: integer:uint32
    "questItemHash": 0,
    // Type: Destiny.Milestones.DestinyPublicMilestoneActivity
    "activity": {
        // Type: integer:uint32
        "activityHash": 0,
        // Type: integer:uint32[]
        "modifierHashes": [
           // Type: integer:uint32
            0
        ],
        // Type: Destiny.Milestones.DestinyPublicMilestoneActivityVariant[]
        "variants": [
           // Type: Destiny.Milestones.DestinyPublicMilestoneActivityVariant
            {
                // Type: integer:uint32
                "activityHash": 0
            }
        ]
    },
    // Type: Destiny.Milestones.DestinyPublicMilestoneChallenge[]
    "challenges": [
       // Type: Destiny.Milestones.DestinyPublicMilestoneChallenge
        {
            // Type: integer:uint32
            "objectiveHash": 0,
            // Type: integer:uint32:nullable
            "activityHash": 0
        }
    ]
}
