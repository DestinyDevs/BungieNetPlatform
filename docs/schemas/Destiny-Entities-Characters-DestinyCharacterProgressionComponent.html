Destiny.Entities.Characters.DestinyCharacterProgressionComponent

A community run wiki for the Bungie.net Platform APIs.

Edit Wiki

Info

This component returns anything that could be considered "Progression" on a user: data where the user is gaining levels, reputation, completions, rewards, etc...

  • Type: object

Properties

Name Type Description
progressions Dictionary<integer:uint32,Destiny.DestinyProgression> A Dictionary of all known progressions for the Character, keyed by the Progression's hash.

Not all progressions have user-facing data, but those who do will have that data contained in the DestinyProgressionDefinition.
factions Dictionary<integer:uint32,Destiny.Progression.DestinyFactionProgression> A dictionary of all known Factions, keyed by the Faction's hash. It contains data about this character's
status with the faction.
milestones Dictionary<integer:uint32,Destiny.Milestones.DestinyMilestone> Milestones are related to the simple progressions shown in the game, but return additional and
hopefully helpful information for users about the specifics of the Milestone's status.
quests Destiny.Quests.DestinyQuestStatus[] If the user has any active quests, the quests' statuses will be returned here.

Note that quests have been largely supplanted by Milestones, but that doesn't mean that
they won't make a comeback independent of milestones at some point.
uninstancedItemObjectives Dictionary<integer:uint32,Destiny.Quests.DestinyObjectiveProgress[]> Sometimes, you have items in your inventory that don't have instances, but still have
Objective information. This provides you that objective information for uninstanced
items.

This dictionary is keyed by the item's hash: which you can use to look up the
name and description for the overall task(s) implied by the objective.
The value is the list of objectives for this item, and their statuses.
