DestinyService / GetItemDetail

Edit Wiki

This page was generated with Wiki Builder. Do not change the format!

Info

Returns information about a unique item instance associated with a character.

  • URI: /Destiny/{membershipType}/Account/{destinyMembershipId}/Character/{characterId}/Inventory/{itemInstanceId}/
  • Method: GET
  • Accessibility: Public
  • Service: DestinyService

Parameters

Path Parameters

Name Description
membershipType A valid Bungie.net membershipType.
destinyMembershipId A valid destinyMembershipId.
characterId A valid characterId that is associated with the given account.
itemInstanceId A valid item instance that is in the character's inventory.

Query String Parameters

Name Description
definitions Include definitions in the response. Use while testing.

JSON POST Parameters

None
