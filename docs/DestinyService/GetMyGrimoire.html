DestinyService / GetMyGrimoire

Info

Returns the Grimoire for the currently signed in account.

  • URI: /Destiny/Vanguard/Grimoire/{membershipType}/
  • Method: GET
  • Accessibility: Private
  • Service: DestinyService

Parameters

Path Parameters

Name Description
membershipType  

Query String Parameters

Name Description
definitions Include definitions in the response; true or false.
flavour Indicates flavour stats should be included with player card data. More testing needed.
single Return data for a single cardId.

JSON POST Parameters

None
