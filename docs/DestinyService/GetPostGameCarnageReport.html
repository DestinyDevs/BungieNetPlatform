DestinyService / GetPostGameCarnageReport

Edit Wiki

Info

Gets the available post game carnage report for the activity ID.

  • URI: /Destiny/Stats/PostGameCarnageReport/{activityInstanceId}/
  • Method: GET
  • Accessibility: Public
  • Service: DestinyService

Parameters

Path Parameters

Name Description
activityInstanceId  

Query String Parameters

Name Description
definitions Include definitions in the response; true or false.

JSON POST Parameters

None
