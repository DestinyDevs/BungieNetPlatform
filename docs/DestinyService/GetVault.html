DestinyService / GetVault

Info

Returns the contents of player's Vault.

  • URI: /Destiny/{membershipType}/MyAccount/Vault/
  • Method: GET
  • Accessibility: Private
  • Service: DestinyService

Parameters

Path Parameters

Name Description
membershipType  

Query String Parameters

Name Description
definitions  
accountId  

JSON POST Parameters

None
