DestinyService / GetDestinyManifest

Edit Wiki

Info

Returns the current version of the manifest as a json object.

  • URI: /Destiny/Manifest/
  • Method: GET
  • Accessibility: Public
  • Service: DestinyService

JSON POST Parameters
