DestinyService / SearchDestinyPlayer

Info

Returns a list of players by username and platform.

  • URI: /Destiny/SearchDestinyPlayer/{membershipType}/{displayName}/
  • Method: GET
  • Accessibility: Public
  • Service: DestinyService

Parameters

Path Parameters

Name Description
membershipType  
displayName A valid display name to search for.

Query String Parameters

None

JSON POST Parameters

None

Example

https://www.bungie.net/Platform/Destiny/SearchDestinyPlayer/2/lowlines/
