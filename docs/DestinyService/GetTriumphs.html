DestinyService / GetTriumphs

Info

Returns a list of triumph sets for a given account.

  • URI: /Destiny/{membershipType}/Account/{membershipId}/Triumphs/
  • Method: GET
  • Accessibility: Public
  • Service: DestinyService

Parameters

Path Parameters

Name Description
membershipType  
membershipId  

Query String Parameters

Name Description
definitions Include definitions in the response; true or false.

JSON POST Parameters

None
