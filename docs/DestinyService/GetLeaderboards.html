DestinyService / GetLeaderboards

Edit Wiki

This page was generated with Wiki Builder. Do not change the format!

Info

Returns leaderboard stats compared against friends. Note you may need to re-authenticate with PSN/Xbox in order to see full rankings.

  • URI: /Destiny/Stats/Leaderboards/{membershipType}/{destinyMembershipId}/
  • Method: GET
  • Accessibility: Private
  • Service: DestinyService

Parameters

Path Parameters

Name Description
membershipType A valid Bungie.net membershipType.
destinyMembershipId A valid destinyMembershipId.

Query String Parameters

Name Description
modes Game modes to return. Comma separated.
statid  
maxtop  

JSON POST Parameters

None

Example

GET https://www.bungie.net/Platform/Destiny/Stats/Leaderboards/2/4611686018428939884/
