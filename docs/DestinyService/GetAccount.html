DestinyService / GetAccount

Info

Returns Destiny account information for the supplied membership.

  • URI: /Destiny/{membershipType}/Account/{membershipId}/
  • Method: GET
  • Accessibility: Public
  • Service: DestinyService

Parameters

Path Parameters

Name Description
membershipType  
membershipId  

Query String Parameters

Name Description
definitions  

JSON POST Parameters

None
