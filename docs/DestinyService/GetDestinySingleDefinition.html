DestinyService / GetDestinySingleDefinition

Edit Wiki

This page was generated with Wiki Builder. Do not change the format!

Info

Returns the specific item from the current manifest a json object.

  • URI: /Destiny/Manifest/{definitionType}/{definitionId}/
  • Method: GET
  • Accessibility: Public
  • Service: DestinyService

Parameters

Path Parameters

Name Description
definitionType The type of definition to return.
definitionId A valid definitionId.

Query String Parameters

Name Description
definitions Include definitions in the response. Use while testing.
version  

JSON POST Parameters

None
