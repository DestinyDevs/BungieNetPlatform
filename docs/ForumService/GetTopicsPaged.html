ForumService / GetTopicsPaged

Edit Wiki

This page was generated with Wiki Builder. Do not change the format!

Info

Returns paged forum results using a given set of filters.

  • URI: /Forum/GetTopicsPaged/{page}/{pageSize}/{group}/{sort}/{quickDate}/{categoryFilter}/
  • Method: GET
  • Accessibility: Public
  • Service: ForumService

Parameters

Path Parameters

Name Description
page The current page to return. Starts at 1.
pageSize The number of results to return. Seems to ignore this and returns 25 results.
group A valid groupId. Use 0 to get all forum posts.
sort How to sort the result. Default is 0 (Default).
quickDate Return results from a predefined date range. Default is 0 (All).
categoryFilter Filter results by a topic category. Default is 0 (None).

Query String Parameters

Name Description
tagstring Filter by tags. Comma separated.

JSON POST Parameters

None

Example

GET https://www.bungie.net/Platform/Forum/GetTopicsPaged/1/1/39966/0/0/0/?tagstring=help
