ActivityService / GetFriendsPaged

Edit Wiki

This page was generated with Wiki Builder. Do not change the format!

Info

Returns a list of friends for a given membershipType.

  • URI: /Activity/Friends/Paged/{membershipType}/{currentPage}/
  • Method: GET
  • Accessibility: Private
  • Service: ActivityService

Parameters

Path Parameters

Name Description
membershipType A valid Bungie.net membershipType.
currentPage The current page to return. Starts at 1.

Query String Parameters

None

JSON POST Parameters

None

Example

GET https://www.bungie.net/Platform/Activity/Friends/Paged/2/1/
