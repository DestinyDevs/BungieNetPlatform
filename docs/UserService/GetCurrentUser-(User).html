UserService / GetCurrentUser (User)

Info

Returns account information for the current user

  • URI: /User/GetBungieNetUser/
  • Method: GET
  • Accessibility: Private
  • Service: UserService

Parameters

Path Parameters

None

Query String Parameters

None

JSON POST Parameters

None
