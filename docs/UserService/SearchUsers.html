UserService / SearchUsers

Edit Wiki

This page was generated with Wiki Builder. Do not change the format!

Info

Returns the user information for the given BungieNet user name query.

  • URI: /User/SearchUsers/
  • Method: GET
  • Accessibility: Public
  • Service: UserService

Parameters

Path Parameters

None

Query String Parameters

Name Description
q A valid Bungie.net username

JSON POST Parameters

None

GET http://www.bungie.net/Platform/User/SearchUsers/?q=DeeJ
